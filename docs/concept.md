# Концепция Sber.Up

## 1. Название идеи

**Sber.Up** - образовательная мультиагентная система, в которой специализированные AI-агенты еженедельно пересобирают индивидуальную учебную траекторию под прогресс студента и меняющиеся запросы рынка.

Идея заменяет статичный диплом непрерывно обновляемым стеком реальных навыков.

## 2. Задача

Sber.Up решает проблему "мертвой" образовательной программы:

- рынок меняется быстрее учебных планов;
- работодателям нужны релевантные кадры здесь и сейчас;
- студент рискует потратить годы на формальный диплом, который уже не соответствует спросу;
- бизнес получает дефицит специалистов при избытке людей с высшим образованием.

## 3. Для кого

Основной пользователь - студент.

Второй контур ценности - бизнес, которому нужны кандидаты с проверенным стеком навыков под реальные задачи.

## 4. Как это работает

У каждого студента есть персональный **AI-Декан** - ансамбль из узкоспециализированных агентов.

### Агент-Диагност

Анализирует входной профиль, цифровой след и когнитивный стиль: как студент решает задачи, пишет код, строит аргументацию и исправляет ошибки.

### Агент Рынка

Парсит актуальные вакансии, коммиты, исследования и тренды. На основе этих сигналов формирует динамическую reward function для целевых навыков.

### Агент-Архитектор

Еженедельно собирает из микромодулей уникальный граф знаний. У студента нет фиксированного факультета. Есть динамический трек: например, не просто "Data Science", а "Инженер по Alignment LLM и распределенным системам".

### Агент-Рефакторинг

Анализирует метрики успеваемости. Если концепт не усвоен, модуль переписывается: меняется симуляция, уровень абстракции, примеры и формат проверки.

Вместо классических сессий - непрерывная валидация через pull request в реальные микро-задачи.

## 5. Роль Сбера

Сбер выступает поставщиком персонализированной образовательной программы:

- подписка для студентов;
- корпоративная витрина для компаний;
- источник реальных микро-задач;
- доверенный бренд для валидации навыков;
- инфраструктура для связки EdTech, HRTech и AI.

## 6. Ценность и эффект

**Для человека:** ниже риск выучиться на невостребованную профессию, потому что траектория постоянно синхронизируется с рынком.

**Для бизнеса:** поток специалистов, подготовленных под актуальные задачи, а не под устаревшие программы.

**Для Сбера:** новая образовательная подписка, кадровый канал и AI-платформа для рынка труда.

## 7. Первый пилот

- 1 000 студентов.
- 4 направления: AI Engineering, Data/ML, Cloud Native, Cybersecurity.
- 12 недель.
- Микро-задачи от внутренних команд и партнеров.
- Метрики: завершение модулей, качество PR, скорость закрытия пробелов, трудоустройство или стажировка.

## 8. Критерии

**Польза:** решает корневую экономическую проблему - дефицит релевантных кадров при избытке формальных дипломов.

**Связь с будущим:** по мере удешевления ИИ персонализированное образование можно масштабировать почти на любого студента.

**Реалистичность:** технологический стек уже есть: LLM, агентные пайплайны, RAG, автопроверки кода, LMS, HR-данные и проектное обучение.

**Оригинальность:** это не очередной онлайн-курс с рекомендациями, а динамический диплом, который пересобирается под рынок и прогресс студента.
