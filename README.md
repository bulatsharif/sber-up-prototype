# Sber.Up

Быстрый статический прототип личного кабинета Sber.Up в яркой сеточной оболочке.

## Что внутри

- `index.html` - кликабельный мок личного кабинета студента.
- `styles.css` - адаптивная верстка под телефон и desktop.
- `script.js` - хардкод-демо выбора курса, пересборки недели и ответов ИИ-Архитектора.
- `docs/` - справочные материалы по идее, не часть основного интерфейса.

## Локальный запуск

```bash
python3 -m http.server 4173
```

Открыть: `http://localhost:4173`

## Концепция по критериям

**Название:** Sber.Up - образовательная мультиагентная система, в которой специализированные AI-агенты еженедельно пересобирают индивидуальную учебную траекторию под прогресс студента и меняющиеся запросы рынка.

**Задача:** решить проблему "мертвой" образовательной программы: рынок требует релевантных кадров сейчас, а выпускники часто получают формальные дипломы без актуального стека навыков.

**Для кого:** студенты, которые хотят учиться под реальные карьерные траектории, и компании, которым нужны специалисты под текущие задачи.

**Как работает:** у каждого студента есть персональный AI-Декан - ансамбль агентов: Диагност, Агент Рынка, Архитектор и Рефакторинг. Они анализируют профиль, рынок, прогресс и каждую неделю собирают новый граф микромодулей.

**Роль Сбера:** поставщик персонализированной образовательной программы в подписочной модели для студентов и в корпоративном контуре для бизнеса.

**Ценность:** студент снижает риск выучиться на невостребованную профессию; бизнес получает кандидатов с проверенным и актуальным стеком навыков.

**Пилот:** 1 000 студентов, 4 динамических направления, 12 недель, микро-задачи от компаний и метрики трудоустройства.
